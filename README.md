# Call-Center-Analysis
Call center performance analysis in Power BI
Dost Tədris Mərkəzində keçdiyim təlim zamanı Power BI vasitəsilə Call Center datası üzərində interaktiv dashboard hazırlamaq imkanı qazandım. Təlim çərçivəsində ilk sərbəst Power BI analizimi Dost Tədris Mərkəzinin təqdim etdiyi Excel faylı əsasında etdim
Hesabatda təhlil etdiyim əsas göstəricilər:
◾ Ümumi müraciətlərin sayı, cavablandırılma faizi, ortalama məmnuniyyət
◾ Həll olunan və olunmayan zənglərin payı
◾ Mövzular və rayonlar üzrə müraciətlərin bölgüsü
◾ Vəzifələr üzrə həll olunan zənglərin faizi
◾ Aylar üzrə həftə içi və həftə sonu müraciət %-i
